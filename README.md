# Java Console Game

This project is a console-based game developed in Java as part of my learning process in object-oriented programming. The goal was to design and implement a complete system from scratch, focusing on game logic, structure, and problem-solving rather than visual aspects.

## Overview

The game is built around a grid-based map where the player navigates through different floors, encounters enemies, collects items, and progresses while managing resources. It includes a basic combat system and different types of entities interacting within the game world.

## Features

* Grid-based map and movement system
* Player with attributes and progression
* Enemy encounters and combat mechanics
* Item handling and basic inventory logic
* Floor-based progression system
* Game loop with clear win/lose conditions

## Technical Approach

The project follows a structured approach by separating responsibilities into different layers:

* `app`: entry point of the application
* `model`: core entities such as player, enemies, items, and map
* `logic`: game mechanics, controllers, and overall flow

This separation allowed me to organize the code more clearly and manage the complexity as the project grew.

## Learning Objectives

The main purpose of this project was to:

* Strengthen my understanding of object-oriented programming
* Practice designing and structuring a medium-sized application
* Improve problem-solving skills through real implementation challenges
* Move from theory to building functional systems

## How to Run

1. Clone the repository
2. Open the project in an IDE such as IntelliJ IDEA
3. Run the main class (`Runner`)

## Limitations

This project is entirely console-based and focuses on logic rather than user interface. Some parts of the design could be further improved as I continue learning more advanced concepts.

## Final Notes

This project represents an important step in my progression as a developer. It reflects not only what I have learned so far, but also my ability to apply those concepts to build something functional and increasingly complex.
